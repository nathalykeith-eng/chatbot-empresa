# Chatbot Empresarial - Mini Proyecto

## Descripción
Proyecto de chatbot web profesional con frontend en React y backend en Node.js + Express. 
Permite a los usuarios hacer preguntas sobre la empresa y recibir respuestas automáticas.

## Tecnologías
- Frontend: React
- Backend: Node.js, Express
- Comunicación API: Axios
- CSS: Inline
